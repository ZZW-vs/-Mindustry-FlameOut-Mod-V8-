# Flame Out

Mindustry熄灭FlameOut模组修改版(V8),转版本且修改了部分内容，用一些其他元素替换了恐怖元素^_^

## 单位

| 单位 | 英文名 | 特点 |
|------|--------|------|
| 共鸣 | Empathy | 可直接生成，拥有 20+ 种随机攻击模式 |
| 冷漠 | Apathy | 击杀后召唤共鸣，多阶段形态切换 |
| 消沉 | Despondency | 最终Boss，多武器协同作战 |
| 世界树 | Yggdrasil | 大型支援单位，触手攻击系统 |

## 隐藏贴图

按 `H` 键打开隐藏贴图菜单，可显示彩蛋剧情中的11张隐藏贴图。

| 按键 | 功能 |
|------|------|
| `H` | 打开隐藏贴图菜单（游戏中生效） |
| 点击贴图 | 将贴图渲染在玩家脚下 |
| 大小/旋转 | 可自定义贴图大小和旋转角度 |
| 清除全部 | 一键移除所有已放置的贴图 |

可用贴图：`main`（主图）、`ball`、`hug`、`cat`、`bunny0-4`（5只兔子）、`flower`、`tree`

## 剧情系统

按 `V` 键启动剧情，支持以下快捷键：

| 按键 | 功能 |
|------|------|
| `Z` | 重置剧情到阶段 0 |
| `X` | 前进到下一阶段 |
| `V` | 启动剧情 |
| `C` | 退出剧情并重启游戏 |
| `B`（按住） | 快进，跳过等待阶段 |
| `F5` | 打开 FlameOut 设置界面 |
| `H` | 打开隐藏贴图菜单 |

> 按 H 打开隐藏贴图菜单，可选择并放置11种隐藏贴图，支持大小、旋转、横向拉伸、纵向拉伸调整

> **注意**：所有快捷键均可在游戏设置的「FlameOut」分类中或按 `F5` 自定义修改，支持恢复默认值。
>
> **单位设置**（F5 → 单位设置）：
> - 冷漠死后生成共鸣：开关控制冷漠单位死亡后是否生成共鸣
> - 共鸣跨地图重生：开关控制共鸣是否在换地图后重生
> - 重置共鸣生成器：清除当前共鸣生成状态，重新开始生成
>
> **剧情设置**（F5 → 剧情设置）：
> - 自动重启游戏：开关控制剧情结束后是否自动重启
> - 自动重启时间：可自定义重启倒计时（默认25秒）
> - 使用原版剧情贴图：开关切换原版/修改版剧情贴图（需重启游戏生效）
>
> **视觉设置**（F5 → 视觉设置）：
> - 血液颜色：8种预设颜色可选（血色/白色/黄色/黑色/蓝色/紫色/橙色/绿色）

## 特性

- **多AI协同**：共鸣单位同时运行攻击AI与移动AI，通过权重随机切换
- **自定义着色器**：黑洞、混沌/秩序、第三冲击、连锁闪电等特效
- **特殊死亡效果**：物理碎裂、血液蒸发、蒸汽效果等
- **Boss战系统**：消沉Boss拥有防空炮、激光、磁轨炮、核弹等多种武器
- **血液颜色可定制**：通过修改 `FlamePal.java` 中的颜色常量调整

## 构建

### 环境要求
- JDK 17
- Gradle（项目自带 wrapper）

### 桌面版
```powershell
.\gradlew jar
```
输出：`build/libs/flameout.jar`

### 手机版（需要 Android SDK）
```powershell
.\gradlew deploy
```
输出同时包含桌面与 Android 兼容的 DEX 格式。

### GitHub Actions
推送代码到 GitHub 后，可通过 Actions 自动构建手机版。

## 项目结构

```
src/flame/
├── unit/           # 单位逻辑
│   ├── empathy/    # 共鸣单位AI与攻击模式
│   ├── weapons/    # 自定义武器
│   └── shifts/     # 冷漠形态切换
├── bullets/        # 自定义子弹
├── effects/        # 特效系统
├── entities/       # 自定义实体
├── graphics/       # 着色器与渲染
├── special/        # 剧情系统
└── audio/          # 音频管理
```
