# Apathy（冷漠）- 单位分析报告

## 一、基本属性

**位置：** `src/flame/unit/ApathyUnitType.java`  
**主要实现：** `src/flame/unit/ApathyIUnit.java`  
**AI系统：** `src/flame/unit/ApathyIAI.java`

### 核心属性：
| 属性 | 数值 | 说明 |
|------|------|------|
| 生命值 | 12,000,000 | 1200万生命值 |
| 护甲 | 60 | 极高的护甲值 |
| 速度 | 0.45 | 移动速度较慢 |
| 碰撞体积 | 75 | 75单位大小 |
| 免疫 | 所有状态效果 | 完全免疫任何状态效果 |

### 单位类型特征：
- **飞行单位**（flying = true）
- **无坠落效果**（fallEffect = Fx.none）
- **无法控制**（controller = unit -> new ApathyIAI()）
- **5种形态切换**（PrismShift, WeakLaserShift, AoEShift, SweepShift, StrongLaserShift）

## 二、形态系统（Shift System）

Apathy 拥有 **5种不同的战斗形态**，可以根据战场情况智能切换！

### 形态列表：

#### 1. Base（基础形态）- 索引0
- **行为**：移动接近敌人核心
- **特征**：无特殊攻击，主要用于移动

#### 2. Small Laser（小激光形态）- 索引1
- **武器**：持续激光射击
- **特性**：
  - 持续追踪最强目标
  - 激光伤害受压力值影响
  - 可切换到其他形态

#### 3. AoE（范围攻击形态）- 索引2
- **武器**：范围弹幕攻击
- **特性**：
  - 扫描周围敌人
  - 发射多个弹幕
  - 对地面单位特别有效

#### 4. Sweep（横扫形态）- 索引3
- **武器**：旋转激光
- **特性**：
  - 激光360度旋转
  - 伤害区域大
  - 适合对付多个目标

#### 5. Strong Laser（强激光形态）- 索引4
- **武器**：超级激光炮
- **特性**：
  - 需要充能时间
  - 造成巨大伤害
  - 激光伤害受压力值影响

## 三、智能AI系统

**文件：** `src/flame/unit/ApathyIAI.java`

### 1. 目标选择系统
- **最强目标（strongest）**：综合评估敌人DPS和距离
- **最近目标（nearest）**：基于距离的目标
- **最近核心（nearestCore）**：敌方基地核心

### 2. 形态切换逻辑
```
评分系统：
- 基线分数：20（基础形态）
- 包围分数：根据周围敌人数量计算
- 集中度分数：敌人分布密度

切换条件：
- 评分高于当前形态
- 使用次数 < 2次
- 冷却时间已过
```

### 3. 压力系统（Stress System）
- **来源**：
  - 受到子弹伤害
  - 周围敌人DPS
  - 盾被击中
- **影响**：
  - 攻击速度提升
  - 形态切换加速
  - 移动速度变化

## 四、防御机制

### 1. 盾牌系统（Shield System）
- **最大盾牌值**：10,000
- **盾牌恢复**：25点/秒（压力越高恢复越快）
- **盾牌眩晕**：4秒（盾牌破碎后）
- **特殊能力**：
  - 反弹低速子弹
  - 改变子弹方向
  - 吸收伤害转压力

### 2. 压力反击机制
```
当受到高伤害攻击时（>500,000）：
1. 立即打断当前攻击
2. 造成血液爆炸特效
3. 眩晕4秒
4. 形态切换速度翻倍
```

## 五、召唤系统（Sentry System）

**文件：** `src/flame/unit/ApathySentryUnit.java`

### 触发条件：
- 盾牌被破坏（shieldStun > 0）
- 生命值低于1/3
- 哨兵数量 < 8

### 哨兵属性：
| 属性 | 数值 |
|------|------|
| 生命值 | 9,000 |
| 大小 | 10 |
| 武器 | 激光 |

### 行为模式：
1. **治疗模式**：为Apathy恢复生命
2. **攻击模式**：攻击最强目标
3. **重新定位**：每3分钟调整位置

## 六、攻击AI详解

### 1. 小激光（Small Laser）
```java
// 持续激光，伤害计算
damage = baseDamage * (1 + stressScaled / 4f)
```

### 2. 大激光（Strong Laser）
```java
// 需要充能
chargeTime = 48帧
// 伤害更高
damage = baseDamage * (1 + stressScaled / 4f)
```

### 3. 范围弹幕（AoE）
```java
// 扫描角度内敌人
// 根据敌人密度调整射速
fireRate = 90 / (1 + stressScaled / 1.5f)
```

### 4. 横扫激光（Sweep）
```java
// 360度旋转
// 旋转速度受压力影响
rotationSpeed = 360度 / lifetime * (1 + stressScaled / 3f)
```

## 七、高级特性

### 1. 关键点系统（Critical Points）
每个形态都有特定的关键点：
- 击中关键点造成10倍伤害
- 触发血液爆炸特效
- 增加压力值

### 2. 音效系统
- 持续音效循环
- 根据激光位置调整音量
- 位置音效计算（多普勒效应）

### 3. 死亡机制
```java
死亡过程：
1. 尖叫音效（45帧后开始）
2. 血液喷溅（持续4分钟）
3. 每16次喷溅发出声音
4. 4分钟后真正死亡
5. 触发Empathy生成！
```

## 九、如何对抗Apathy

### 弱点：
1. **速度慢**：可以风筝战术
2. **盾恢复需要时间**：集中火力破盾
3. **关键点**：避免击中特定角度
4. **形态切换**：切换时可能有破绽

### 建议策略：
1. 使用快速单位进行游击
2. 高DPS武器优先破盾
3. 避免正面硬刚
4. 等待形态切换时进攻
